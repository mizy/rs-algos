# Rust WebAssembly algos package

## dependencies
- [rust](https://www.rust-lang.org/tools/install)
- [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/)

## build
```bash
make build
```

## test
```
make test
```

## algos
+ graph
  - apsp
  - bfs
  - cnc
  - sssp
+ layouts
  - nforce_3d (a 3d force directed layout algorithm)
  - radial_force

## pack wasm
```bash
make build
```
